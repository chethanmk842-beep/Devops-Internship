Overview

This project implements a DevOps workflow aimed at improving code reliability through automated testing and continuous code quality validation.

Tasks Completed
	•	Created and configured a CI pipeline using GitHub Actions
	•	Integrated SonarQube to perform static code analysis on the repository
	•	Enabled automatic triggering of builds and scans on code push events
	•	Reviewed and fixed code quality issues identified during analysis
	•	Maintained successful pipeline execution with improved code standards

Tools & Technologies Used
	•	Git & GitHub
	•	GitHub Actions
	•	SonarQube
	•	Docker 

Outcome

The workflow ensures continuous validation of code quality, helping maintain a stable and production-ready codebase through automated DevOps practices.
